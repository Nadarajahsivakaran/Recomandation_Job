ML based Job Recommendation

Job Recommendation System using Machine Learning and Flask.

This Flask application serves as a job recommendation system powered by a machine learning model. It allows users to input their education background, GPA, interests, work experience, and preferences for job location and remote work. The system then predicts and recommends suitable career goals based on this input data. The predictions are made using a pre-trained Random Forest classifier, and categorical data is encoded using LabelEncoders loaded from pre-saved models.

Features:

  • Flask framework for web application development.
  • Machine learning model trained on job recommendation data.
  • Integration of scikit-learn for machine learning tasks.
  • Dynamic dropdown menus for selecting education, field of study, interests, relevant work experience, skills, job location, and remote work preferences.
  • Responsive UI design for seamless user experience across devices.
  • Input validation to ensure data integrity and accuracy of recommendations.
  • Ability to reset form inputs and fetch new recommendations.
  • Encapsulation of business logic in Python functions for modular development.
